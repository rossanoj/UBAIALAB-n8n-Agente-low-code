# UBAIALAB-n8n-Agente-low-code
Trabajo practico para el curso Diseño e implementación de agentes de IA low code/no code

Diseño de un flujo de trabajo para control de la gestion de un proyecto para los eventos de crear una rama, solicitar un pull request y aprobarlo para su integracion

La primer etapa es un flujo sencillo que informa por mail cada uno de los pasos, desde la creacion de una rama, la solicitud de aprobacion (pull request), y posteriormente la aceptacion o rechazo del pull request, considerando la opcion de eliminarlo o cancelarlo. 
Estos pasos quedan registrados en una tabla de la base de datos.
